⚡ OSC / WebSocket Bridge
Browsers can't send UDP OSC directly. You need a local WebSocket→OSC bridge.

Quick start with Node.js:
npm i -g osc-bridge && osc-bridge --ws 8080 --udp 57120

Or run Open Stage Control as a relay (it includes a WS server).
With Max/MSP: use Node4Max or CNMAT odot externals.
With Python: pip install python-osc websockets + custom bridge script.
